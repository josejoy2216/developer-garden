# Developer Garden 🌱

A VS Code extension that turns your work into a living garden. Every project and skill grows as a plant — fed by your real work (time, saves, commits, tests). Achievements add decorations, wildlife moves in, and a timeline keeps your whole journey. Nothing ever dies, nothing nags you, and all data stays 100% local.

## Install & run

```bash
npm install
npm run compile
```

Open the folder in VS Code and press **F5** — a new VS Code window opens with the extension running. Open any project, start coding, and check the 🌱 icon in the activity bar.

## Play

- Tap a plant → water it (💧, once a day) or change its species (🌿)
- Tap the grass → a wildflower blooms
